# Youtube iOS App

Esta es una aplicación de YouTube para iOS desarrollada en Swift. La app presenta una interfaz que replica las funcionalidades básicas de YouTube, incluyendo un feed de videos, secciones de trending y suscripciones, un reproductor de video integrado con controles de reproducción/pausa, y un menú de configuración con opciones de cuenta. Utiliza arquitectura MVC con servicios API personalizados, vistas de colección para el diseño de contenido, y carga asíncrona de imágenes para optimizar el rendimiento.
